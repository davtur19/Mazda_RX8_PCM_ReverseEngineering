CRITICAL IMPORTANCE NOTICE
================================================================================

The Oil Metering Pump (OMP) is ESSENTIAL for Wankel rotary engine survival:

FUNCTION: Injects lubricating oil directly into the combustion chambers through
          dedicated solenoid-driven pump independent of fuel injection system.

APEX SEAL PROTECTION: Rotary engine apex seals (rotating sealing vanes) require
  continuous oil film to prevent metal-to-metal contact with rotor housing.
  Apex seal failure = catastrophic engine damage, engine seizure, total loss.

CONSEQUENCES:
  - INSUFFICIENT OIL (low duty cycle): Carbon buildup, apex seal wear, scoring,
    eventual catastrophic failure. Silent failure mode (no immediate symptoms
    until seal fails under load).

  - EXCESS OIL (high duty cycle): Blue/white smoke, fouled spark plugs (oil
    fouling), carbon deposit formation, possible emission control failures.

DESIGN: Unlike 4-stroke engines where cylinder walls retain oil film, rotary
  combustion chambers are large open spaces. Oil must be injected fresh each
  combustion event to maintain seal protection.

function oil_temp_burn_control() {
  // INITIALIZATION

  // Cache frequently accessed ROM table addresses
  rom_timing_corr_table = &timing_correction_3EE0A;
  rom_cold_start_table = &cold_start_enrichment_3EEB8;

  // Cache RAM register addresses
  ram_enable_flag = 0xFFFFB178;
  ram_output_duty = 0xFFFFB17C;
  ram_last_output = 0xFFFFB180;
  ram_oil_temp_counter = 0xFFFFCB38;
  ram_load_correction = 0xFFFFB0E4;
  ram_output_limit = 0xFFFFB187;  // Safety limit tracking

  // ENABLE CHECK (Step 1)

  if (!read_byte(ram_enable_flag)) {  // Check enable flag
    write_word(ram_last_output, 0x0000);  // Force output to zero
    call_table_lookup(rom_cold_start_table, index=0);  // Abort sequence
    return;  // Exit early if disabled
  }

  // OIL TEMPERATURE TRACKING (Step 2)

  oil_temp_counter = read_byte(ram_oil_temp_counter);  // 0xFFFFCB38

  // Saturated increment of thermal counter
  // Used to track oil temperature state without actual temperature sensor
  // (assumes oil temperature correlates with engine runtime)
  updated_counter = saturated_add_u16(
    current=read_word(ram_last_output),
    increment=1,
    max_value=0xFFFF
  );
  write_word(ram_last_output, updated_counter);

  // BASE DUTY LOOKUP (Step 3A)

  // Primary lookup: timing_correction table with oil temperature index
  // This table address 0x3EE0A is referenced but actual table not found in
  // normal calibration area. May be dynamic or computed value.
  base_duty_float = table_lookup(
    table=rom_timing_corr_table,
    index=0xFFFF8124  // Index computed from engine state
  );

  // THERMAL COMPENSATION (Step 3B)

  // Secondary table: cold start enrichment
  // Adds additional oil when engine is cold
  thermal_comp = table_lookup(
    table=rom_cold_start_table,
    index=0xFFFF8124
  );

  thermal_comp_result = thermal_comp * loaded_value;

  // LOAD CORRECTION (Step 3C)

  // Read load-based correction factor from RAM
  load_correction = read_float(ram_load_correction + 0x0E4);

  // Add load correction to base duty
  duty_with_load = base_duty_float + load_correction;

  // Apply second thermal compensation
  thermal_comp2 = table_lookup(
    table=rom_cold_start_table,
    index=0xFFFF8124
  );

  // INTERPOLATION & NORMALIZATION (Step 4)

  counter_value = read_word(ram_last_output);

  if (counter_value > 0) {
    // Normalize duty by counter (effectively scales duty by frequency)
    interpolation_weight = 1.0;
    normalized_duty = linear_interpolate(
      value=counter_value,
      weight=interpolation_weight,
      min=0.0,
      max=1.0
    );

    // Final duty = interpolated value
    final_duty = normalized_duty / counter_value;
  } else {
    // Counter at zero: use raw duty
    final_duty = 0.0;
  }

  write_float(ram_output_duty, final_duty);  // Output to 0xFFFFB17C

  // SAFETY BOUNDS CHECK (Step 5)

  counter_value = read_word(ram_last_output);
  safety_limit = read_word(0x3EE0A + 0x10);  // Safety upper bound

  if (counter_value >= safety_limit) {  // Sanity check
    // Clamp to safe range
    write_word(ram_last_output, 0x0000);

    // Retrieve safe min/max values
    safe_min = read_float(0x3EE0A + 0x30);
    safe_max = read_float(0x3EE0A + 0x34);

    // Force select between min/max
    final_duty = select_between_bounds(
      current_duty=read_float(ram_output_duty),
      min=safe_min,
      max=safe_max
    );

    write_float(ram_output_duty, final_duty);

    // Emergency disable sequence
    call_table_lookup(rom_cold_start_table, index=0xFFFF8114);
    call_table_lookup(rom_cold_start_table, index=0xFFFF8124);
  }

  // THERMAL STATE TRACKING (Step 6)

  current_temp_index = read_byte(ram_output_limit);

  if (current_temp_index != oil_temp_counter) {

    if (oil_temp_counter == 1) {
      // Cold start condition
      base_table_index = lookup_table(0xFFFF8114);
      thermal_correction = lookup_table(0xFFFF811C);
    } else {
      // Normal operation
      base_table_index = lookup_table(0xFFFF811C);
      thermal_correction = load_float(0xFFFF811C);
    }

    // Apply thermal correction
    thermal_comp = lookup_table(0xFFFF8114);
    final_duty = 0.0;  // Reset

    write_float(ram_output_duty, final_duty);
    call_table_lookup(rom_cold_start_table, 0xFFFF8124);
  }

  // STATE MACHINE UPDATE
  write_byte(ram_output_limit, oil_temp_counter);
}

7. CALIBRATION STRATEGY & TUNING PARAMETERS
================================================================================

STOCK OMP CALIBRATION (RX-8 S1 EU N3J1EL):

PRIMARY DUTY TABLE (Oil_Metering_By_Load):
  Structure: 17 RPM points × 19 load points
  Typical Values: 3% (idle) to 28% (WOT/high RPM)
  Characteristic: Non-linear, increases with both RPM and load

  OPTIMIZATION TARGETS:
    ✓ Idle (500-1000 RPM, 0% load): 3-5% duty
      → Too low: carbon buildup, apex seal wear
      → Too high: blue smoke, fouled plugs

    ✓ Cruise (2000-3000 RPM, 20% load): 8-12% duty
      → Balances protection with emissions

    ✓ WOT (6000-8000 RPM, 100% load): 25-40% duty
      → Maximum oil delivery for high friction conditions

    ✓ Cold Start (below 0°C): +5-10% additional duty
      → Compensates for cold oil high viscosity

SECONDARY DUTY TABLE (Oil_Metering_By_Throttle):
  Structure: 4 throttle points
  Typical Values: 0% (idle) to 23% (WOT acceleration)
  Characteristic: Transient enrichment during load changes

  OPTIMIZATION TARGETS:
    ✓ Throttle sweep enrichment: Smooth acceleration without seal starvation
    ✓ Engine braking: Maintain minimum oil during deceleration
    ✓ Launch control: Extra oil during hard acceleration

TEMPERATURE COMPENSATION:
  Cold Start Table (cold_start_enrichment_3EEB8):
    - Adds extra duty when oil is below 60-80°C
    - Gradual decay as engine warms up
    - Typical extra duty: 5-15% added to base

  Warm-up Transition:
    - Oil temp counter increments each cycle
    - Triggers switch from cold-start to normal table
    - Hysteresis prevents chattering

SAFETY LIMITS (Table_2D___319_Check_DataType):
  - Maximum duty: 50-60% (prevents solenoid stall/coil damage)
  - Minimum duty: 2-3% (prevents seal starvation)
  - Emergency cutoff: Duty > 80% = fault condition

8. CRITICAL CONDITIONS & SAFETY MECHANISMS
================================================================================

COLD START PROTECTION:
  ✓ Oil viscosity highest when cold → highest friction
  ✓ Extra oil injection for first 5-30 seconds after start
  ✓ Counter-based thermal tracking (no temperature sensor)
  ✓ Automatic transition to normal duty after warmup
  ✓ Prevents apex seal damage during cold-start high-friction regime

HIGH-RPM PROTECTION:
  ✓ Base duty increases with engine speed
  ✓ Higher RPM = higher apex seal speeds = more friction
  ✓ Centrifugal forces increase seal loading
  ✓ Extra oil injection at high RPM (6000+) prevents seal failure

FULL-LOAD/ACCELERATION PROTECTION:
  ✓ Throttle-based transient enrichment
  ✓ Load-based duty increase from Oil_Metering_By_Load table

Symptoms:
  • Difficult cold starts (especially below 0°C)

// OMP Control Main Entry Point
// Called once every 10-50ms from main engine control loop
void oil_temp_burn_control_1FC32(void) {

  // Static/cached register addresses (optimized by compiler)
  static const u32 ROM_TIMING_CORR = 0x3EE0A;      // Lookup table #1
  static const u32 ROM_COLD_START = 0x3EEB8;       // Lookup table #2
  static const u32 RAM_ENABLE = 0xFFFFB178;        // Enable flag
  static const u32 RAM_DUTY_OUT = 0xFFFFB17C;      // Duty output (float)
  static const u32 RAM_COUNTER = 0xFFFFB180;       // State counter
  static const u32 RAM_LOAD_CORR = 0xFFFFB0E4;     // Load correction
  static const u32 RAM_THERM_IDX = 0xFFFFB187;     // Thermal state
  static const u32 RAM_OIL_TEMP = 0xFFFFCB38;      // Oil temp counter

  // Local variables
  u8 oil_temp;
  u16 counter;
  float duty;
  float thermal_comp;
  float load_factor;

  // STEP 1: Check if OMP is enabled
  if (readB(RAM_ENABLE) == 0) {
    // OMP disabled - set zero output
    writeW(RAM_COUNTER, 0);
    table_lookup_thermal_abort(ROM_COLD_START, IDX_0);
    return;
  }

  // STEP 2: Read and track oil temperature state
  oil_temp = readB(RAM_OIL_TEMP);           // Signed temperature data

  // Saturated add for counter (tracks thermal accumulation)
  counter = readW(RAM_COUNTER);
  counter = saturated_add(counter, 1, 0xFFFF);
  writeW(RAM_COUNTER, counter);

  // STEP 3: Look up base duty from ROM
  duty = table_lookup(ROM_TIMING_CORR, IDX_ENGINE_STATE);

  // STEP 4: Add thermal compensation (more oil when cold)
  thermal_comp = table_lookup(ROM_COLD_START, IDX_ENGINE_STATE);
  duty += thermal_comp;

  // STEP 5: Apply load-based correction
  load_factor = readF(RAM_LOAD_CORR);
  duty += load_factor;

  // STEP 6: Second thermal comp lookup
  thermal_comp = table_lookup(ROM_COLD_START, IDX_ENGINE_STATE);

  // STEP 7: Interpolate and normalize duty
  counter = readW(RAM_COUNTER);
  if (counter > 0) {
    float weight = 1.0;
    float norm = linear_interp(counter, weight, 0.0, 1.0);
    duty = norm / (float)counter;
  } else {
    duty = 0.0;
  }

  writeF(RAM_DUTY_OUT, duty);  // Write to output

  // STEP 8: Safety bounds checking
  if (counter >= SAFETY_LIMIT_HIGH) {
    // Overflow condition - clamp to safe range
    writeW(RAM_COUNTER, 0);
    float safe_min = readF(ROM_TIMING_CORR + 0x30);
    float safe_max = readF(ROM_TIMING_CORR + 0x34);
    duty = clamp(readF(RAM_DUTY_OUT), safe_min, safe_max);
    writeF(RAM_DUTY_OUT, duty);
    table_lookup_abort_sequence(ROM_COLD_START);
  }

  // STEP 9: Thermal state machine
  u8 prev_therm_idx = readB(RAM_THERM_IDX);
  if (prev_therm_idx != oil_temp) {
    // Thermal regime changed
    if (oil_temp == COLD_START_INDEX) {
      // Enter cold start mode
      float cs_duty = table_lookup(ROM_COLD_START, IDX_COLD);
      float cs_corr = table_lookup(ROM_COLD_START, IDX_CORR_COLD);
    } else {
      // Normal warm operation
      float norm_duty = table_lookup(ROM_COLD_START, IDX_NORMAL);
      float norm_corr = readF(RAM_LOAD_CORR);
    }
    writeB(RAM_THERM_IDX, oil_temp);  // Update thermal state
  }

  return;  // Exit to main loop
}

// Support function: Lookup ROM calibration table
float table_lookup(const u32 table_addr, u16 index) {
  // Simplified: actual implementation does 1D/2D/3D interpolation
  // Returns float value from calibration table
  float* table_ptr = (float*)table_addr;
  return table_ptr[index];
}

// Support function: Linear interpolation
float linear_interp(float value, float weight, float min_out, float max_out) {
  // Linear interpolation: out = min + (max - min) * weight
  return min_out + (max_out - min_out) * weight;
}

// Support function: Saturated add with bounds
u16 saturated_add(u16 current, u16 increment, u16 max_val) {
  u32 temp = (u32)current + (u32)increment;
  if (temp > max_val) return max_val;
  return (u16)temp;
}

GENERAL TUNING PHILOSOPHY:

The stock OMP calibration is tuned for:
  1. Reliability (maximize seal life)

RECOMMENDED ADJUSTMENTS:

✗ NOT RECOMMENDED:
