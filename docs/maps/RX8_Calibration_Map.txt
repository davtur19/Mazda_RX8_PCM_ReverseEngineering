10. SUMMARY AND NOTES

Calibration Area Overview:
  Total ROM Size: 4,352 bytes (0x6CF00 to 0x7E000)
  Number of Calibration Tables: 1,214 items (including axis arrays)
  Primary Table Types: Fuel (25%), Ignition (20%), Idle (15%), Misc (40%)

Data Type Distribution:
  - u8 (8-bit unsigned): Small scalar values, flags, counts
  - u16 (16-bit unsigned): Axis points, RPM values, temperatures
  - u32 (32-bit unsigned): Larger scalar values
  - float (32-bit IEEE): Temperature thresholds, rev limits, calibration constants

Scaling Patterns:
  RPM Values: Typically stored as u16, scaled by 0.25 or multiplied by 25
  Temperature: Stored as u16 ADC or u32 float, offset -40C
  Throttle/Load: 0-255 ADC or 0-100% scaled
  Degrees: Often 0-255 = 0-360 degrees mapping

Critical Float Values Identified:
  Rev_Limit @ 0x6D54C = 1175232512 (raw u32) = ~8000 RPM (physical limit)
  Cold_Rev_Limit_Threshold @ 0x6D53C = 1101004800 = ~20C threshold

Common Axis Patterns:
  Fuel/Ignition tables: 80-byte RPM axis (20 points at 25 RPM spacing)
  Load tables: 72-byte load axis (18-20 points)
  Temperature: 28-40 bytes (7-10 points from -40C to +130C)

Known Table Clusters:
  Leading Ignition: 0x6D6B0-0x6D8B0 (large 360-byte data tables)
  Fuel Open Loop: 0x71BD8-0x72964 (extensive fuel mapping)
  OMP Control: 0x79034-0x79264 (oil pump lookup tables)
  Sensor Scaling: 0x6F8EC-0x70050 (thermistor/voltage conversions)

Reverse Engineering Notes:

  - Float type inference based on value ranges and typical ECU patterns
  - Axis sizes often vary by 4-byte increments suggesting alignment padding
