================================================================================
MAZDA RX-8 S1 PCM — COMPLETE FIRMWARE FUNCTION MAP
ROM: N3J1EL (60E1D400) — EU 6-port 231hp MT, 2004-2008
CPU: Renesas SH7055 — Big-endian, 16-bit instructions
Total Functions: ~2,789 (all named, no sub_ prefixes remain)
Code Range: 0x00000040 - 0x0006C000+
Calibration Data: 0x6CE00 - 0x7E000
================================================================================

NOTE: ~780+ functions have NOT been individually audited. Function names marked
⚠ should be treated as provisional. See RX8_Audit_Report.txt for full verification
results.

    Messages (from Ghidra cross-validation, standard Mazda CAN IDs):
    TX from ECU:
    - ID 0x201 — Engine RPM (bytes 0-1) + vehicle speed (4-5) + throttle (6) + status (7)
    - ID 0x203 — Engine status pack
    - ID 0x215, 0x231, 0x240, 0x250, 0x251 — TX data packs
    - ID 0x420 — MIL/check-engine status

    - ID 0x212 — RX main handler
    - ID 0x216 — RX with plausibility check, fault eval, timer
    - ID 0x430, 0x47, 0x4B0, 0x4B1, 0x4C0 — RX unpack handlers

    Values 0xCA/0xDC in disassembly are internal mailbox/queue identifiers,
    NOT the actual CAN arbitration IDs on the bus.

    Description: High-speed CAN (FlexCAN) communication handler. Manages
    periodic and event-driven message transmission via standard Mazda 11-bit
